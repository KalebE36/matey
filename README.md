# Running the project 
1. Compile the application using our make file in root dir
```
make compile
```
2. We have the ability to run three different peers that are specified in our `cfg/PeerInfo.cfg`
```
make run-peer-0 # Peer 1
make run-peer-1 # Peer 2
make run-peer-2 # Peer 3
```

# Formatting
1. Run formatter using makefile
```
make format
```
